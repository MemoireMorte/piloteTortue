Pilote Tortue

Cette application permet de contrôler une tortue Jeulin T3 via une connexion série. Je n'ai pas de tortue T2 pour tester. Les cartes sont classées par couleur. Il semble que chaque couleur corresponde à un type de carte différent, même si ce n'est pas évident pour certaines cartes.
Le présent projet est en cours de développement et peut être retrouvé sur github : https://github.com/MemoireMorte/piloteTortue
